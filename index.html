Blockchain App Web3 Service

Project
Previous Next

Smart Contract

This app facilitates sending transactions on chain. Users can send or receive Ethereum with metamask connected.

All Transaction are sent or received using Ethereum testnet Ropsten. Users will need to install metamask and use their public ETH address to send transaction to another eth wallet on Ropsten Network. If wallet is not installed, app will automatically display a message to install metamask.

This app is made using React Js and tailwind css for front end and Solidity for smart contract. The project is deployed to github pages.

Features

App UI is made with React Js and Tailwind CSS

Responsive Form with validation added

Connect UI with Smart Contract

Smart Contract is written in Solidity language and connected to UI with web3 library web3.js

Develpment was facilitated with hardhat enviroenment and i used Alchemy node for smart contract deployment

Smart Contract interacts with Client using ABI.
